# Jayce Scanner v3.3.1

Solana token chart scanner with AI-powered setup detection.

## Features
- 5M timeframe chart analysis
- Vision AI + Pattern matching scoring system
- Three-tier alerts: FORMING (40+) / VALID (55+) / CONFIRMED (70+)
- 45-minute Vision rejection cooldown
- 223 training charts from real trade data
- Daily vision budget cap (350/day)

## Stack
- Python + Playwright (headless Chromium)
- Anthropic Vision API
- Telegram alerts
- Railway deployment
